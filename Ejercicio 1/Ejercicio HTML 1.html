¿Que es HTML?

HTML, sigla en ingles de HyperText Markup Language (lenguaje de marcas de hipertexto), hace referencia al lenguaje de marcado para la elaboración de paginas web. Es un estándar que sirve de referencia del software que conecta con la elaboración de paginas web en sus diferentes versiones, define una estructura básica y un codigo (denomindao código HTML) para la definición de contenido de una pagina web, como texto, imagenes, videos, juegos, entre otroe.
